# Dataset Explorer Web App

This is an interactive data analytics website where users can upload any CSV dataset and explore insights automatically.

## Features

- Upload CSV dataset
- Preview dataset in table format
- View dataset summary (rows & columns)
- Automatic chart generation for all columns
- Detects numeric columns and shows trend charts
- Filter data based on column values

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
- PapaParse

## Purpose of Project

This project was built to practice data analysis thinking, data visualization, and basic web development skills.

## How to Use

1. Upload a CSV file  
2. View preview table and summary  
3. Explore charts  
4. Use filter to analyze specific data
